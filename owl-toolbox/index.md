# OWL Toolbox – Privacy Policy

**Last updated:** February 2026

## Overview

OWL Toolbox is a Windows desktop application developed and distributed by Owl Automata.  
The application is provided **free of charge** and is available both outside of the Microsoft Store and through the Microsoft Store.

Your privacy is important to us. This document explains how OWL Toolbox handles user data.

## Data Collection

OWL Toolbox **does not collect any personal data**.

Specifically, the application:

- Does **not** collect personal information  
- Does **not** collect usage analytics  
- Does **not** track user behavior  
- Does **not** access location data  
- Does **not** collect identifiers or diagnostic data  

## Data Transmission

OWL Toolbox:

- Does **not** send any data to external servers  
- Does **not** communicate with cloud services  
- Does **not** use third-party analytics, telemetry, or advertising services  

All functionality is performed **locally on the user’s device**.

## Data Storage

- No personal or identifiable data is stored by the application  
- Any configuration or settings created by the user remain **locally on the device**  

## Third-Party Services

OWL Toolbox does **not** integrate with or depend on any third-party services that collect or process user data.

## Children’s Privacy

OWL Toolbox does not collect any data and is therefore compliant with children’s privacy regulations. The application is safe to use by users of all ages.

## Changes to This Privacy Policy

If OWL Toolbox gains new features that affect privacy (such as optional online functionality), this privacy policy will be updated accordingly.

Any changes will be published in this repository.

## Contact

If you have any questions or concerns regarding this privacy policy, you may contact us via our GitHub organization page or repository issues.
