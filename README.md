# OWL Automata – Privacy Policies

This repository contains the official privacy policies for applications developed by **OWL Automata** (founded in 2023).

Each application has its own folder with a dedicated privacy statement used for:
- Google Cloud OAuth verification  
- Microsoft Azure AD / Microsoft Identity Platform verification  
- End-user transparency and legal compliance  

All documents in this repository are publicly accessible and served through GitHub Pages.

---

## Repository Structure

```text
privacy/
├── orochi/
│   └── index.md
├── another-app/
│   └── index.md
└── ...
```


Each `index.md` file contains the privacy policy specific to that application, including:
- What data the app accesses  
- How the data is used  
- Storage and security practices  
- Google Limited Use compliance  
- Microsoft Graph API disclosures  
- Contact information for privacy inquiries

## Data Protection Commitment

OWL Automata is committed to:
- Minimizing the collection of personal data  
- Ensuring transparency in the use of user information  
- Complying with Google API Services User Data Policy (including Limited Use)  
- Complying with Microsoft Developer Policies and Microsoft Graph API requirements  
- Storing all authentication tokens securely on-device where applicable  

This repository provides the public documentation required by these compliance frameworks.
